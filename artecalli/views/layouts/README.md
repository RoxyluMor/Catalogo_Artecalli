# Catálogo ArteCalli

## Descripción
Sistema web desarrollado para la gestión y visualización de un catálogo de productos de onix y marmol.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- PHP
- Xampp / MySQL

## Instalación y ejecución

1. Clonar repositorio:
   git clone https://github.com/usuario/catalogo-artecalli.git

2. Abrir en Visual Studio Code

3. Configurar base de datos:
   - Importar archivo SQL incluido

4. Ejecutar servidor local:
   - Usar XAMPP 

5. Acceder desde navegador:
   http://localhost/catalogo-artecalli
